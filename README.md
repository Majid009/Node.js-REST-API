# Node.js-REST-API
In this REST API all CRUD operations are performed using all convetions of REST Architechure. This is simple api to manage courses.

# Dependencies 
1- Express
2- body-parser

# Courses REST API details

# 1-Get List of All Courses
URL: /api/courses/
Http Method:GET
Response Type:JSON

# 2-Get a Course with specific id
URL: /api/courses/2   ( 2 is course id)
Http Method:GET
Response Type:JSON

# 3-Add New Course
URL: /api/courses/add
Http Method:POST
Response Type:JSON

 Parametes:-
    id (int)
    name (string) 
    fee (string)
    
# 4-Delete Course
URL: api/courses/delete/2  ( 2 is course id)
Http Method:DELETE
Response Type:JSON
